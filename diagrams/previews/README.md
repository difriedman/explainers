# Diagram previews

PNG exports of each `.excalidraw` source in the parent folder. These are referenced from the top-level [README](../../README.md) so visitors can preview diagrams without opening Excalidraw.

## Export conventions

- **Format:** PNG (transparent background OK; white background preferred for embeddability).
- **Scale:** 2x (for crisp rendering on Hi-DPI displays).
- **Filename:** identical to the source file but with a `.png` extension.
  - e.g. `Fabric overview.excalidraw` → `Fabric overview.png`

## How to export

1. Open the `.excalidraw` file at [excalidraw.com](https://excalidraw.com/) (File → Open) or in the Excalidraw VS Code extension.
2. **Export image** (top-left menu) → PNG, scale 2x, background ✅.
3. Save into this folder using the matching name.

After updating a source diagram, re-export the PNG so the README preview stays in sync.
